# AI Enabled Workflow Adoption Config Tool and Dashboard

This is a static prototype for identifying which AI tools support a process.

Open `index.html` in a browser, or serve the folder locally and visit the local URL.

## What is included

- Default four-step process shown left to right.
- Two to three supporting activities under each step.
- Two to three AI tools under each step.
- Checkboxes to mark which tools support each step.

The earlier assessment and utilization scoring concepts are intentionally out of the primary experience so the first prototype focuses on clean data capture.
